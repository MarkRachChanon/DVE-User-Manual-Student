## 1. การสมัครสมาชิกและเข้าสู่ระบบ

### 1.1 การสมัครสมาชิก (Register)

1. เข้าหน้าแรกของระบบ แล้วกดปุ่ม **"สมัครสมาชิก"**
2. เลือกแท็บ **"นักเรียน / นักศึกษา"** (เป็นค่าเริ่มต้นอยู่แล้ว)
3. กรอกข้อมูลตามขั้นตอน:

   **ข้อมูลส่วนตัว:**
   | ช่อง | คำอธิบาย |
   |------|----------|
   | คำนำหน้า | เลือก นาย, นาง, นางสาว, ด.ช., ด.ญ. |
   | ชื่อ | กรอกชื่อจริง |
   | นามสกุล | กรอกนามสกุล |
   | รหัสนักศึกษา | กรอกรหัสนักศึกษาของคุณ เช่น 6512345678 |

   **ข้อมูลสถานศึกษา (สำคัญมาก!):**
   | ช่อง | คำอธิบาย |
   |------|----------|
   | วิทยาลัย | เลือกวิทยาลัยที่คุณศึกษาอยู่ |
   | ระดับชั้น | เลือกระดับ ปวช. / ปวส. / ป.ตรี |
   | สาขาวิชา | เลือกสาขาวิชาของคุณ (**ต้องเลือกระดับชั้นก่อน**) |

   > **⚠️ สำคัญ:** เลือกวิทยาลัย ระดับชั้น และสาขาวิชาให้ **ตรงตามความเป็นจริง** เพราะระบบจะใช้ข้อมูลนี้คัดกรองชั้นเรียนที่คุณสามารถเรียนได้

   **ข้อมูลบัญชี:**
   | ช่อง | คำอธิบาย |
   |------|----------|
   | Username | ตั้งชื่อผู้ใช้อย่างน้อย 3 ตัวอักษร |
   | อีเมล | อีเมลสำหรับรับ OTP และการแจ้งเตือน |
   | รหัสผ่าน | อย่างน้อย 6 ตัวอักษร |
   | ยืนยันรหัสผ่าน | กรอกรหัสผ่านซ้ำอีกครั้ง |

4. กดปุ่ม **"สมัครสมาชิก"**

### 1.2 การยืนยันอีเมล (OTP)

หลังสมัครสำเร็จ ระบบจะพาคุณไปหน้ายืนยันอีเมล:

1. เปิดอีเมลของคุณ ค้นหาอีเมลจากระบบ DVE
2. คัดลอก **รหัส OTP 6 หลัก** ที่ได้รับ
3. กรอกรหัสในหน้ายืนยัน → กดยืนยัน

> **💡 ไม่พบอีเมล?**
> - ตรวจสอบโฟลเดอร์ **"จดหมายขยะ (Spam/Junk)"**
> - รหัส OTP มีอายุ **15 นาที** — กด **"ส่งรหัสใหม่"** หากหมดอายุ

### 1.3 การรอการอนุมัติบัญชี

- ในบางวิทยาลัย แอดมินอาจต้อง **ตรวจสอบและอนุมัติบัญชี** ก่อน
- หน้าจอจะแสดงข้อความ **"รอการอนุมัติ"** — ให้รอจนกว่าแอดมินจะอนุมัติ
- หากรอนานเกินไป → ติดต่อเจ้าหน้าที่วิทยาลัย

### 1.4 การเข้าสู่ระบบ (Login)

1. กรอก **อีเมล** และ **รหัสผ่าน**
2. กดปุ่ม **"เข้าสู่ระบบ"**
3. ระบบจะนำไปยังหน้า **Dashboard นักเรียน** โดยอัตโนมัติ

> **⚠️ ล็อกอินผิดหลายครั้ง?**
> ระบบจะ **ล็อคบัญชีชั่วคราว** และแสดงตัวนับเวลาถอยหลัง ให้รอจนหมดเวลาแล้วลองใหม่

### 1.5 ลืมรหัสผ่าน

1. กดลิงก์ **"ลืมรหัสผ่าน?"** ในหน้า Login
2. กรอกอีเมลที่ลงทะเบียนไว้
3. ตรวจสอบอีเมลและตั้งรหัสผ่านใหม่ผ่านลิงก์ที่ได้รับ

## 2. หน้า Dashboard นักเรียน

หลังจาก Login สำเร็จ คุณจะเห็นหน้า **"ภาพรวมการเรียน"** ประกอบด้วย:

### 2.1 สถิติภาพรวม (4 การ์ดหลัก)

| สถิติ | คำอธิบาย |
|-------|----------|
| **ชั้นเรียนของฉัน** | จำนวนชั้นเรียนที่ลงทะเบียนอยู่ |
| **เรียนจบแล้ว** | จำนวนวิชาที่เรียนครบ 100% |
| **แบบทดสอบที่ผ่าน** | จำนวนแบบทดสอบที่ผ่านเกณฑ์ / จำนวนครั้งที่ทำ |
| **คะแนนเฉลี่ย** | คะแนนเฉลี่ยรวมจากแบบทดสอบทั้งหมด |

### 2.2 ความคืบหน้าการเรียน

- แสดง **เปอร์เซ็นต์ความคืบหน้าโดยรวม** (เนื้อหา + แบบทดสอบ)
- แสดงจำนวน: เนื้อหาที่จบ, แบบทดสอบที่ทำ, ชั่วโมงเรียน

### 2.3 ผลการทำแบบทดสอบ

- แสดงจำนวนแบบทดสอบที่ทำทั้งหมด
- จำนวนที่ผ่านเกณฑ์
- คะแนนเฉลี่ย

### 2.4 ชั้นเรียนล่าสุด

- แสดงชั้นเรียนที่ลงทะเบียนพร้อม **Progress Bar** ความคืบหน้า
- คลิกที่ชั้นเรียนเพื่อเข้าเรียนต่อ
- ปุ่ม **"ค้นหาชั้นเรียน"** สำหรับลงทะเบียนวิชาใหม่

## 3. การเข้าร่วมชั้นเรียน

นักเรียนสามารถเข้าร่วมชั้นเรียนได้ **4 วิธี:**

### 3.1 แท็บ "ชั้นเรียนของฉัน"

- แสดงชั้นเรียนที่ลงทะเบียนและได้รับอนุมัติแล้ว
- คลิกที่การ์ดเพื่อเข้าเรียน

### 3.2 แท็บ "รออนุมัติ"

- แสดงชั้นเรียนที่ลงทะเบียนแล้วแต่ **ยังรอครูอนุมัติ**
- เมื่อครูอนุมัติ ชั้นเรียนจะย้ายไปแท็บ "ชั้นเรียนของฉัน" อัตโนมัติ

### 3.3 แท็บ "เข้าร่วมด้วยรหัส"

สำหรับเข้าร่วมชั้นเรียนที่ครูแจ้งรหัสให้:

1. กรอก **รหัสชั้นเรียน (Class Code)** หรือวาง **Invite Link** ที่ได้รับจากครู
2. กดปุ่ม **"ตรวจสอบข้อมูลชั้นเรียน"**
3. ระบบจะแสดงข้อมูลชั้นเรียน (ชื่อ, ผู้สอน, สถานศึกษา)
4. กดปุ่ม **"ยืนยันการเข้าร่วม"**

> **💡 วิธีรับรหัส:** สอบถามจากครูผู้สอนประจำวิชา หรือตรวจสอบในอีเมลคำเชิญ

### 3.4 แท็บ "ลงทะเบียนเพิ่มเติม"

- ค้นหาและเรียกดูชั้นเรียนที่เปิดรับสมัคร (Internal/Public)
- ใช้ช่องค้นหาเพื่อหาชั้นเรียนที่สนใจ
- กดปุ่ม **"ลงทะเบียนเรียน"** ที่ชั้นเรียนที่ต้องการ
- ถ้าชั้นเรียนตั้งค่า Auto Approve → เข้าเรียนได้ทันที
- ถ้าชั้นเรียนตั้งค่า Require Approval → ต้องรอครูอนุมัติก่อน

## 4. การเรียนบทเรียน

### 4.1 การเข้าเรียน

1. คลิกที่ชั้นเรียนจาก Dashboard หรือหน้ารายการชั้นเรียน
2. เลือกบทเรียนที่ต้องการเรียน
3. ระบบจะเปิดหน้าเรียนแบบเต็มจอ

### 4.2 ระบบล็อคบทเรียน (Content Lock) ⚠️

- บทเรียนจะถูกจัดเรียงตามลำดับ 1, 2, 3...
- คุณ **ไม่สามารถข้ามไปเรียนบทถัดไปได้** หากบทก่อนหน้ายังไม่เสร็จ
- ต้องดูเนื้อหาครบทุกชิ้นและผ่านแบบทดสอบ (ถ้ามี) จึงจะปลดล็อคบทถัดไป

### 4.3 เงื่อนไขการจบเนื้อหา

| ประเภทเนื้อหา | เงื่อนไขการจบ |
|---------------|--------------|
| **วิดีโอ** | ต้องรับชมอย่างน้อย **98%** ของความยาวทั้งหมด |
| **เอกสาร** | ต้องเปิดดูจนครบ |
| **รูปภาพ** | ต้องเปิดดู |
| **ข้อความ** | ต้องอ่าน |
| **ลิงก์** | ต้องเข้าชม |

### 4.4 กฎความปลอดภัย (Focus Protection) 🛡️

- **วิดีโอจะหยุดเล่นทันที** หากคุณ:
  - เปลี่ยนไปเปิดแท็บอื่น
  - ย่อหน้าต่างเบราว์เซอร์
  - เปิดแอปอื่นทับ
- กฎนี้เพื่อให้มั่นใจว่าคุณรับชมเนื้อหาจริงๆ

### 4.5 ระบบช่วยจำ (Resume)

- หากคุณออกจากระบบระหว่างเรียน → เมื่อกลับมาเข้าเรียนใหม่ ระบบจะเปิดไปที่ **บทเรียนที่เรียนค้างไว้** โดยอัตโนมัติ
- ไม่ต้องจำว่าเรียนถึงไหนแล้ว

## 5. การทำแบบทดสอบ

### 5.1 ประเภทแบบทดสอบ

| ประเภท | เมื่อไหร่ | จุดประสงค์ |
|--------|----------|-----------|
| **Pretest** | ก่อนเรียน | วัดความรู้เบื้องต้น |
| **Posttest** | หลังเรียน | วัดผลสัมฤทธิ์การเรียน |
| **Practice** | ระหว่างเรียน | ฝึกฝนทบทวน (ทำซ้ำได้) |

### 5.2 ขั้นตอนการทำแบบทดสอบ

1. เข้าไปที่บทเรียน → กดปุ่ม **"เริ่มทำแบบทดสอบ"**
2. อ่านโจทย์และเลือก/กรอกคำตอบ
3. กดปุ่ม **"ส่งคำตอบ"** เมื่อทำเสร็จ

### 5.3 ระบบจับเวลา ⏰

- หากข้อสอบมีการจำกัดเวลา → จะมีนาฬิกานับถอยหลังที่มุมจอ
- **เมื่อเวลาหมด ระบบจะส่งคำตอบอัตโนมัติทันที** ในสถานะที่ทำค้างไว้
- ❗ ไม่สามารถขอเวลาเพิ่มได้

### 5.4 การดูผลคะแนน

- หลังส่งคำตอบ ระบบจะแสดง **ผลคะแนนทันที** (ยกเว้นข้ออัตนัยที่ต้องรอครูตรวจ)
- แสดงข้อมูล: คะแนนที่ได้, เปอร์เซ็นต์, สถานะผ่าน/ไม่ผ่าน
- หากเปิดให้แสดงเฉลย → สามารถดูเฉลยได้หลังส่งคำตอบ

### 5.5 การสอบใหม่

- หากทำคะแนน **ไม่ถึงเกณฑ์** (เช่น ต่ำกว่า 60%)
- กดปุ่ม **"ทำแบบทดสอบอีกครั้ง"** (ถ้าครูอนุญาตให้สอบใหม่ได้)
- บางแบบทดสอบอาจจำกัดจำนวนครั้งที่ทำได้

### 5.6 การเช็คพิกัด GPS

- บางแบบทดสอบอาจเปิดการเช็คพิกัด GPS
- ระบบจะขออนุญาตเข้าถึงตำแหน่งของคุณ → กด **"อนุญาต"**
- ข้อมูลตำแหน่งจะถูกบันทึกเพื่อยืนยันว่าคุณทำข้อสอบจากสถานที่ที่กำหนด

## 6. การดูความคืบหน้าและผลการเรียน

### 6.1 หน้า "ความคืบหน้าของฉัน" (My Progress)

เข้าได้จากเมนู **"ความคืบหน้า"** ใน Sidebar ซ้าย:

- แสดงรายการชั้นเรียนทั้งหมดที่ลงทะเบียน
- แต่ละชั้นเรียนแสดง **เปอร์เซ็นต์ความคืบหน้า** พร้อม Progress Bar
- สีเขียว = เรียนจบ (100%) / สีแดง-ส้ม = ยังเรียนไม่จบ

### 6.2 รายละเอียดความคืบหน้า

คลิกที่ชั้นเรียนเพื่อดูรายละเอียด:

| ข้อมูล | คำอธิบาย |
|--------|----------|
| จำนวนเนื้อหาที่จบ | เช่น 8/12 เนื้อหา |
| จำนวนแบบทดสอบที่ทำ | เช่น 3/5 แบบทดสอบ |
| เปอร์เซ็นต์รวม | ความคืบหน้าโดยรวม |
| ชั่วโมงเรียน | เวลาที่ใช้เรียนทั้งหมด |

### 6.3 ดูประวัติการทำแบบทดสอบ

- เข้าไปที่แบบทดสอบ → กดปุ่ม **"ประวัติการทำ"**
- แสดงทุกครั้งที่ทำ พร้อมคะแนนและสถานะผ่าน/ไม่ผ่าน

## 8. คำถามที่พบบ่อย (FAQ)

### ❓ สมัครสมาชิกแล้วแต่ล็อกอินไม่ได้?

- ตรวจสอบว่า **ยืนยันอีเมล (OTP)** เรียบร้อยแล้ว
- ตรวจสอบว่าบัญชี **ได้รับการอนุมัติ** จากแอดมินแล้ว (บางวิทยาลัย)
- หากบัญชีถูกระงับ → ติดต่อแอดมินวิทยาลัย

### ❓ ลงทะเบียนชั้นเรียนแล้วแต่เข้าเรียนไม่ได้?

- ชั้นเรียนอาจตั้งค่าให้ **ต้องรอการอนุมัติจากครู**
- ตรวจสอบแท็บ **"รออนุมัติ"** ในหน้าชั้นเรียน
- แจ้งครูผู้สอนเพื่อให้อนุมัติ

### ❓ ไม่เห็นชั้นเรียนที่ต้องการ?

- ชั้นเรียนอาจเป็นแบบ **Private** ที่จำกัดเฉพาะสาขา/ระดับชั้น
- ตรวจสอบว่าข้อมูล **สาขาวิชาและระดับชั้น** ของคุณถูกต้อง
- ลองใช้ **รหัสชั้นเรียน** หรือ **Invite Link** จากครู

### ❓ ข้ามบทเรียนไม่ได้?

- ระบบตั้งค่าให้เรียน **ตามลำดับ** — ต้องจบบทก่อนหน้าก่อน
- ตรวจสอบว่าดูเนื้อหาครบทุกชิ้นและผ่านแบบทดสอบ (ถ้ามี)

### ❓ วิดีโอหยุดเล่นกลางคัน?

- ระบบมี **Focus Protection** — วิดีโอจะหยุดเล่นเมื่อสลับแท็บหรือย่อหน้าต่าง
- ให้กลับมาที่หน้าเรียนและกดเล่นต่อ

### ❓ เวลาทำข้อสอบหมดแล้ว?

- ระบบจะ **ส่งคำตอบอัตโนมัติ** ในสถานะที่ทำค้างไว้
- ข้อที่ยังไม่ได้ตอบจะได้ 0 คะแนน

### ❓ ทำข้อสอบไม่ผ่าน ทำอีกได้ไหม?

- ขึ้นอยู่กับการตั้งค่าของครู — บางแบบทดสอบอนุญาตให้ทำซ้ำได้
- หากมีปุ่ม **"ทำแบบทดสอบอีกครั้ง"** แสดงว่าทำซ้ำได้
